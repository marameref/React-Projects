# Node.js Core Fundamentals Checkpoint

This project demonstrates the core capabilities of the Node.js runtime, including the File System (FS), HTTP networking, and third-party package integration via NPM.

## 📂 Project Structure
- **hello-world.js**: Basic console output.
- **server.js**: A local HTTP server running on Port 3000.
- **file-system.js**: Asynchronous file creation and data retrieval.
- **password-gen.js**: Secure random password generation.
- **email-sender.js**: Automated email dispatch using SMTP.

## ⚙️ Installation
1. Clone the repository.
2. Install dependencies: `npm install`
3. Configure your `.env` file with `EMAIL_USER` and `EMAIL_PASS`.

## 🛡️ Security
This project follows security best practices by using environment variables to mask sensitive credentials.
