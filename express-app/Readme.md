# Working Hours Web App

This is a **Node.js/Express** application that uses custom middleware to restrict user access based on the time of the request.

---

## Features
* **Time-Restricted Access:** Only accessible Monday to Friday, from 9:00 to 17:00.
* **Dynamic Rendering:** Built with the **EJS** template engine.
* **Styled UI:** Simple, clean CSS navigation and layout.

---

## Prerequisites
* **Node.js** installed on your machine.

---
